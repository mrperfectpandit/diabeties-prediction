Diabetes Predictor









This Machine Lerning web application is used to predict weather the patient have diabeties or not , on the behalf of some input feature i.e.
Number of Pregnancies
Glucose in ml/dl
Blood pressure of the patient in mmhg
Skin Thickness of patient in mm
Insulin level of patient in IU/ml
Body Mass Index (BMI) in kg/m.sq.
Diabetes pedigree function
Age of the patient


Overview

The dataset was downloaded from kaggle competition and perform little bit feature engineering like handling NAN values than make a model using ensemble technique i.e Random forest regressor
this web application use Flask api and deployed on Heroku
